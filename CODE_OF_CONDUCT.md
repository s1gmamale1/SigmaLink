# Code of Conduct

SigmaLink adopts the [Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct, and expects every contributor and maintainer to abide by it.

## Reporting

We do not maintain a separate code-of-conduct email. To report a violation, open a private security advisory at:

> https://github.com/s1gmamale1/SigmaLink/security/advisories/new

This channel is private; only project maintainers can read it.

## Enforcement

Maintainers will review every report, respond to the reporter in a reasonable timeframe, and take whatever corrective action they judge appropriate. Possible actions range from a private warning, to temporary loss of contribution privileges, to a permanent ban from project spaces.

The full canonical text of the policy lives at the link above. Refer to it for the definitions of acceptable behavior, unacceptable behavior, scope, and enforcement guidelines.
