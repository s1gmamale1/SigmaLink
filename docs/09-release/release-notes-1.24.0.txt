SigmaLink v1.24.0 — 2026-05-26
==============================

Per-room polish + accessibility — frontend Stage 4, the FINAL stage of the
Apple-grade frontend roadmap (Stages 1-4 now complete). Built by 3 parallel
worktree-isolated coders, lead-merged with the full gate in main (incl. the
whole tests/e2e/ dir + a prefers-reduced-motion runtime check).


Added — global reduce-motion safety-net
---------------------------------------

- A single @media (prefers-reduced-motion: reduce) reset in index.css collapses
  animation + transition durations to ~0 app-wide. One block covers everything
  previously unguarded: the notification bell pulse, the Jorvis Orb's inline
  keyframes, all Radix animate-in/out (dialog / dropdown / popover / tooltip /
  sheet), and every animate-pulse / ping / spin / bounce. Uses 0.01ms (not
  none) so animationend/transitionend still fire. Verified at runtime:
  computed animation-duration is 0.01ms under reduce vs 1s normally.


Added — landmarks
-----------------

- Skip-to-main-content link (first focusable element) + an #main landmark on the
  main content region; aria-label="Sidebar" on the sidebar landmark.


Changed — standardized empty / loading / error states
-----------------------------------------------------

- Reuse the shared EmptyState / ErrorBanner / Spinner components consistently:
  - Memory: window.alert removed (a browser modal) -> dismissible ErrorBanner.
  - OperatorConsole: RPC errors were silently swallowed -> now surface in a
    dismissible ErrorBanner.
  - Bare "Loading…" text -> Spinner (Memory graph, Review DiffView/ConflictsTab,
    Skills tab + marketplace).
  - Bare empty text -> EmptyState (Review SessionList, Memory graph).
  - SigmaBench gained the workspace-guard EmptyState every other room had.


Changed — keyboard + accessible names
-------------------------------------

- The Swarm RoleRoster agent card was keyboard-dead (role="button" with no
  tabIndex/handler) -> now tabIndex={0} + Enter/Space activation.
- aria-label added to icon-only buttons: Browser address bar (back/forward/
  reload/stop/home) + tab strip, Tasks column "Add task", Memory "Create note",
  Operator "Rename mission".
- Hand-rolled Task drawers gained aria-labelledby + Escape-to-close +
  return-focus. (Full Tab-containment focus-trap noted as a follow-up.)
