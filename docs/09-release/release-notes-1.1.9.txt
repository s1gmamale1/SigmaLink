SigmaLink v1.1.9 — Perf paired refactor + file-size sweep + lint zero

Two coordinated swarms shipped this release. PR #3 (Codex + Claude
finalizer) landed the perf paired refactor + CI hardening + lint
wave 32→0. A second 3-coder swarm landed the file-size targets
that closed out the v1.1.9 backlog. Zero behavioural changes. Zero
broken contracts.

Highlights
==========

* **Lint baseline: 32 problems → 0**. React-compiler structural
  family fully resolved (set-state-in-effect deferred to next
  tick, render-time `Math.random()` in sidebar skeleton replaced
  with deterministic sizing, `shared/rpc.ts` `any` typed, narrow
  file-level immutability disables on MemoryGraph + Constellation
  intentional-mutable-ref surfaces).

* **useAppStateSelector<T> + useAppDispatch**. New hooks backed by
  `useSyncExternalStore` so consumers can subscribe to a
  per-selector slice of state instead of the full context.
  CommandRoom, CommandPalette, SwarmRoom, OperatorConsole all
  migrated — the 27-consumer rerender fan-out on every dispatch is
  gone for these four hot rooms.

* **Precomputed slices: `sessionsByWorkspace` + `swarmsByWorkspace`**.
  Reducer maintains both on `ADD_SESSIONS` / `MARK_SESSION_EXITED` /
  `REMOVE_SESSION` / `SET_SWARMS` / `UPSERT_SWARM` /
  `MARK_SWARM_ENDED`. Consumers read the O(1) lookup instead of
  re-filtering `state.sessions` on every render.

* **Three big files split under the 500-LOC budget**:

    - `swarms/factory.ts` 713 → 396 LOC. Private spawn helpers
      (`spawnAgentSession`, `pickCoordinatorId`, `buildExtraArgs`,
      `loadAgentSession`, plus a new `materializeRosterAgent`
      dedupe) moved to `factory-spawn.ts` (344 LOC). Public surface
      unchanged.

    - `assistant/runClaudeCliTurn.ts` 709 → 348 LOC. Stateless
      emit layer moved to `runClaudeCliTurn.emit.ts` (186 LOC).
      Tool routing + Ruflo trajectory + readline-loop dispatchers
      moved to `runClaudeCliTurn.trajectory.ts` (193 LOC). Public
      surface unchanged; `__resetProbeCache` + `__resetActiveChildren`
      preserved for test hooks.

    - `renderer/app/state.tsx` 553 → 97 LOC. The 14 IPC-event
      listener effects extracted into four custom hooks under
      `state-hooks/`: `use-session-restore.ts` (142 LOC),
      `use-workspace-mirror.ts` (65 LOC), `use-live-events.ts`
      (140 LOC), `use-exited-session-gc.ts` (49 LOC), plus a
      shared `parsers.ts` (163 LOC) with a `runRefreshOnEvent`
      helper that dedups four near-identical refresh-on-event
      effects. Provider is now ~150 LOC of pure orchestration.

* **CI / test infra hardening**:

    - GitHub Actions `cache-dependency-path` fixed (was stale).
    - pnpm install switched to `--no-frozen-lockfile --ignore-scripts`
      + explicit `node node_modules/electron/install.js` after.
    - `pnpm run coverage` + `@vitest/coverage-v8` + baseline
      thresholds (lines 22 / statements 21 / functions 21 /
      branches 18).
    - ShellCheck CI step for `app/scripts/install-macos.sh`.
    - `app/coverage/` + `docs/06-test/` added to .gitignore.
    - `.claude` added to ESLint globalIgnores (Claude Code's
      local harness was leaking a non-source-tree lint error into
      the build).
