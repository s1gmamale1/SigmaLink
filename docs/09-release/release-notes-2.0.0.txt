SigmaLink v2.0.0 — 2026-06-07
=============================

The first tagged major release. v2.0.0 bundles the entire 6-phase
next-phase ROADMAP plus its full long-tail, the Phase 0 release-blocker
crisis fix, Phases 7-10, the Phase 9 orchestration/memory-surfacing arc,
and two operator-found in-place resume fixes — all staged on `main`
(untagged) since 2026-06-03 and finalized after live operator
verification on 2026-06-07.

See CHANGELOG.md [2.0.0] for the complete, itemized record.


Headline themes
---------------

  post-crash launch lockout, and workspace-loss all resolved.
- One Apple motion language (P2) + the notification & soundscape
  system (P3).

- Phases 7-10: worktree GUI + multi-workspace-same-dir, premium Jorvis

Final cycle (2026-06-06/07) — folded into this tag
--------------------------------------------------

- Phase 9 — Orchestration & memory surfacing (#125 9A, #126 9B):
  a persistent "Sigma" right-rail panel (Canvas to-dos + live swarm
  token delta · Review/routing trace), the memory graph made <=1 click
  from any room, a global "Automations" room (Telegram + nightly digest),
  per-conversation NAMED sessions (migration 0036 + panes.rename), and a
  per-session Artifacts view. The roadmap's feature work is now complete.

- Fixed — in-place pane session resume (#127, #128), found in live
  operator testing and invisible to the e2e smoke:
  * #127 stopped panes in an in-place (shared-cwd) workspace from
    collapsing onto one session — the operator's own running CLI session
    or a sibling pane's — by never falling back to a "continue-latest"
    flag on a null/ghost id (start fresh instead; the policy gemini
    already used).
  * #128 stopped the "starts a new session each reopen" loop by
    capturing + persisting each claude pane's real --session-id on the
    fresh-fallback, so the next reopen resumes by a real id.
  Both Opus-reviewed; operator-confirmed live before tagging. codex
  panes deliberately stay fresh-each-reopen (safe; their disk-scan
  capture races in a shared cwd) — deterministic codex resume via its
  stdout banner is a noted follow-up.
