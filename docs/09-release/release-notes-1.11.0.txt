Assistant identifier rename: Sigma → Jorvis
-------------------------------------------

Renames the assistant's internal code identifiers from "Sigma" to
"Jorvis", completing the code-level half of the v1.8.0 label-only
rename. This is Cluster A of the W-6 sweep; the DB-table + cross-sync
rename (Cluster B) ships separately as its own carefully-gated packet.

- Folders/files (history-preserving git mv): sigma-assistant/ →
  jorvis-assistant/ (17 files); SigmaRoom.tsx → JorvisRoom.tsx;
  SigmaTabPlaceholder.tsx → JorvisTabPlaceholder.tsx.
- Identifiers: useSigma* → useJorvis*; buildSigmaSystemPrompt →

  → mcp-jorvis-host-server.cjs.
- Window event sigma:sigma-jump-to-message → jorvis:jump-to-message
  (in-process, renamed atomically).
