## Core definition
"Agentic coding is a software development methodology where AI agents operate as autonomous teammates on your engineering team."

## Core principles
- Specialized roles (Architect, Frontend, Backend, QA) per system prompt.
- Shared context: all agents share task board and accumulated findings; Task Knowledge records discoveries.
- Structured task lifecycle: todo → in-progress → in-review → complete; no completion without human approval.
- Human-in-the-loop: every task reviewed.

## Distinction
"Multiple specialized agents work concurrently on different parts of the same project, coordinating through a shared task board and context system."
