## Core definition
"Vibe coding is building software by describing what you want in natural language and letting AI agents write the code."

## Four-step workflow
1. Describe your intent.
2. AI agents generate the code.
3. Review and iterate.
4. Ship with confidence.

## Comparison table (verbatim)
| Aspect | Traditional | Vibe Coding |
|--------|-------------|-------------|
| Primary Input | Manual code writing | Natural language descriptions |
| Builder Role | Writer and debugger | Architect and reviewer |
| Iteration Speed | Hours to days per feature | Minutes to hours per feature |
| Barrier to Entry | Years of practice required | Clear communication is core skill |
| Context Management | Mental model + docs | AI agents maintain automatically |
| Quality Control | Code review + testing | Structured workflow with human approval gates |
