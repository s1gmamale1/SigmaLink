## Feature list (verbatim)
- Multi-provider AI: Anthropic, OpenAI, Google, Groq, Mistral, Bedrock, OpenRouter; mid-session model switching.
- Built-in agents: vibe, build, plan, architect, ship + custom agents.
- MCP integration with any MCP server (incl. BridgeMind MCP).
- Full tool system: file ops, shell exec, web search, code search, sub-agent orchestration, batch ops.
- Session management: persistent conversations with auto-compaction, sharing, forking, export.
- Cross-platform: CLI, web, native desktop (macOS/Windows/Linux).

## Built-in agents (verbatim)
| Agent | Purpose |
|-------|---------|
| vibe | Default NL coding agent balancing speed/quality |
| build | Implementation-focused; writes code, runs tests, iterates |
| plan | Creates detailed technical plans before coding |
| architect | Designs system architecture and data models |
| ship | End-to-end agent: plans, builds, tests, deploys |

Plus custom agents scoped to a project.
