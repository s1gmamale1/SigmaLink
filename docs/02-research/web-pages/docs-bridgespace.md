## Feature list (verbatim)
- Multi-pane terminals — Split your workspace into 1–16 terminal panes.
- Integrated code editor — Syntax highlighting, language detection, file watching.
- File browser — Navigate project files directly in the sidebar.
- AI agent workflows — Kanban board for task management.
- Workspace templates — Pre-built layouts for 1, 2, 4, 6, 8, 10, 12, 14, or 16 terminal panes.
- 25+ themes — Dark-first design with Void, Neon Tokyo, Synthwave, Dracula, more.
- Auto-updates — Seamless background updates.

## Terminal features
- Splitting (horizontal/vertical)
- Search (Cmd+F macOS, Ctrl+F Win/Linux)
- Context menu (copy, paste, clear, split)
- Drag and drop file paths
- Image preview support
- Scroll-to-bottom indicator

## Keyboard shortcuts (macOS, verbatim)
- Cmd+T — New workspace tab
- Cmd+W — Close current tab
- Cmd+P — Quick Open (file search)
- Cmd+F — Search in terminal
- Cmd+D — Split pane
- Cmd+1 through Cmd+9 — Switch to tab by number
