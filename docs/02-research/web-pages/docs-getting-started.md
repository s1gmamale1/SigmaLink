## Prerequisites
- BridgeMind account at bridgemind.ai.
- AI coding agent: Cursor, Claude Code, Windsurf, Codex CLI, or BridgeCode.
- Node.js 18+.

## Steps (paraphrased)
- Step 1: Sign in with Google OAuth.
- Step 2: Settings → API Keys → Generate New Key (with descriptive name).
- Step 3: Follow /docs/mcp#installation.
- Step 4: Have agent call create_project.
- Step 5: Have agent call create_task with instructions and optional knowledge.
- Step 6: Agent picks up task → in-progress → in-review → complete.
