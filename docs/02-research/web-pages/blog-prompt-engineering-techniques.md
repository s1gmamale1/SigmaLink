## Four Pillars
1. Intent Clarity — what, who-for, success looks like; outcomes over impl.
2. Constraint Scoping — tech choices, architecture, performance, scope.
3. Context Loading — existing code, conventions, structure.
4. Iteration Patterns — review holistically, describe problems, refine in layers.

## Advanced patterns
- System-Context-Task Model: role + existing code + requirements.
- Negative Space: specify what NOT to implement.
