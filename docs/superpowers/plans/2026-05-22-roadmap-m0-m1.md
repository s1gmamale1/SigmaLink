# Roadmap M0 + M1 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Ship M0 (debt closure) + M1 "Glanceable swarm" (per-pane info bar, agent index, density, visible swarm chat log) from the BridgeMind-competitive roadmap.

**Architecture:** M1 is renderer-heavy and rides on existing data layers — `session.branch/cwd/worktreePath` (already on `AgentSession`), `swarms.get/list/tail/sendMessage/broadcast` RPCs (already allowlisted), and `GridLayout.tsx`'s existing drag-resize. New work = surface this data in UI + one small git-status RPC + one `.claude` hook. Spec: `docs/superpowers/specs/2026-05-22-bridgemind-competitive-roadmap-design.md`.

**Tech Stack:** Electron 30 · React 19 · TS 5.9 · vitest · Radix UI · lucide-react · better-sqlite3.

**Verified during planning:** C-3 drag-resize ALREADY exists in `GridLayout.tsx` (so C-3 = density only). C-4/C-2 data layers ALREADY exist (RPCs above). PaneHeader already holds branch/model in a tooltip.
