### 3. Write `docs/09-release/release-notes-1.4.7.txt`

```
SigmaLink v1.4.7 — sync across machines, offline Windows voice, notifications

The big one
-----------
Cross-machine session sync ships. Your Sigma conversations, pane history,
memory, and tasks now follow you between devices via a private git remote
you own. End-to-end encrypted with age — SigmaLink never sees your data,
and the encryption key never leaves your machine.

Setup is a 5-step wizard:
  1. Welcome screen
  2. Pick a git remote (private GitHub repo recommended)
  3. Connect (SSH key or token)
  4. Generate + save a recovery mnemonic
  5. First sync test

Two machines, same remote, same mnemonic → automatic sync every 30s.

What else
---------
• Offline Windows voice (SAPI5). Air-gapped Windows users get parity with
  the macOS Speech.framework path.
• Notifications + top-right bell. Three sources: pane exits, swarm
  broadcasts, Sigma tool errors. Click-through navigation.
• Provider auto-install prompt. Try to use kimi without installing it →
  modal with the install command + "Install now in a pane" button.
• Windows auto-update (opt-in). Self-hosted feed via electron-updater.
• OpenCode SQLite direct read. Faster session picker.

Quality
-------
CI is fully green. All 11 Playwright e2e tests pass. The 3 deferred tests
from v1.4.6 and the 2 pre-existing timeouts are closed.

Install / upgrade
-----------------
• macOS: curl -sSL https://sigmalink.dev/install.sh | bash
• Windows: iwr https://sigmalink.dev/install.ps1 -UseB | iex
• In-app: Settings → Updates → Check for updates

Migration
---------
All four new features (sync, notifications, auto-install, Windows auto-update)
default off. No behavior change for existing users until they opt in.

⚠ If you enable sync, SAVE THE RECOVERY MNEMONIC. Without it, your sync
data is permanently undecryptable.

GitHub: https://github.com/s1gmamale1/SigmaLink/releases/tag/v1.4.7
```
