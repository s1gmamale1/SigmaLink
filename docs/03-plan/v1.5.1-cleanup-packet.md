## Shared dispatch protocol (all 3 clusters)

Each agent:

1. **Setup**:
   ```bash
   cd <worktree>/app
   pnpm install --no-frozen-lockfile
   node node_modules/electron/install.js   # v1.4.7 @electron/rebuild ABI fix
   ```

2. **Read context**: this brief + `CHANGELOG.md` `[1.5.0]` Deferred sections + `master_memory.md` Phase 35/36/37 deferred tables. Each caveat has explicit source-PR + reviewer reasoning documented there.

3. **Implementation**: per cluster scope above. If a caveat hits a hard blocker (e.g. dependency installed but API mismatch), STOP and report — do NOT defer to v1.5.2 without escalation; mark as blocker in completion message.

4. **Verification gate**:
   ```bash
   pnpm exec tsc -b --pretty false
   pnpm exec eslint .
   pnpm exec vitest run
   pnpm run build
   node scripts/build-electron.cjs
   ```

5. **PR flow**: stage + commit per-caveat; push branch + open PR titled `fix(v1.5.1-<cluster>): <summary>`. PR body must list each caveat addressed (or explicitly deferred-again).

6. **Completion message**: PR URL + `git diff main --stat` + gate result + per-caveat status.
